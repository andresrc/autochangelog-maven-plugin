autochangelog:attach

Full name:

com.cloudbees.maven.plugins:autochangelog-maven-plugin:1.0:attach

Description:

Attaches a changelog .json as a secondary artifact to the build.

Attributes:

  • Requires a Maven project to be executed.
  • Executes as an aggregator plugin.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: package.

Required Parameters

Name Type Since Description
connectionType String - Allows the user to choose which scm connection to use when connecting to the scm. Can either be "connection" or "developerConnection".
Default value is: connection.

Optional Parameters

Name Type Since Description
aggregateTestVersions boolean - If true and the version is not a test version, then the previous version will be determined by looking for the previous non-test version
Default value is: true.
User property is: autochangelog.aggregateTestVersions.
encoding String - (no description)
User property is: project.build.outputEncoding.
finalName String - (no description)
User property is: project.build.finalName.
outputDirectory File - Directory to create the changes.json file.
User property is: project.build.directory.
passphrase String - The passphrase (used by java svn).
User property is: passphrase.
password String - The user password (used by svn and starteam protocol).
User property is: password.
previousVersion String - (no description)
User property is: autochangelog.previousVersion.
privateKey String - The private key (used by java svn).
User property is: privateKey.
skip boolean - Whether to skip to automatic changelog or not.
Default value is: false.
User property is: autochangelog.skip.
tagBase String - The url of tags base directory (used by svn protocol).
User property is: tagBase.
testVersions String - A comma separated list of case insensitive strings that identify test versions
Default value is: alpha,beta,rc.
User property is: autochangelog.testVersions.
username String - The user name (used by svn and starteam protocol).
User property is: username.
