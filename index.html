Autochangelog Maven Plugin

CloudBees' Autochangelog Maven plugin is used when you want to capture the significant changes since the most recent release of a project.

Goals Overview

The Autochangelog Maven Plugin has the following goals.

  • autochangelog:attach Captures and attaches the changes.json.
